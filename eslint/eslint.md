# 简介
[ESLint](http://eslint.org/)

- 支持React、ES6
- 不同目录支持不同规则


## 中文参考
<https://github.com/Jocs/ESLint_docs/>
<http://www.jianshu.com/p/1682b91756b1>


## 安装流程

```bash
# 需要Node环境
npm install -g eslint

# 根据编辑器 或者 IDE 安装插件
```

## rules

```js
"quotes": [2, "double"]
# 第一部分是规则名
# 第二部分是表示级别：
- "0"/"off" -不验证；
- "1"/"warn" -警告；
- "2"/"error" -错误
```
